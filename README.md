# Instructions

## Clone repo
```
git clone git@github.com:ArnaudDauguen/cpp_cours.git
cd cpp_cours
```
**OU**
```
git clone https://github.com/ArnaudDauguen/cpp_cours.git
cd cpp_cours
```

## Compile
```
g++ -Wall -g src/*.cpp -Iinclude -o bin/tp.exe
```

## Launch
```
./bin/tp.exe
```
